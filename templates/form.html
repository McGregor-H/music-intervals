The interval program

The purpose of this process is to extrapolate entire sequences

of related pitch material from a small set of music intervals .

Since the largest interval needed to reach any pitch from any

other pitch is a tri-tone, this program utilzes only these intervals.


[only input numbers 1 through 6]

1=minor 2nd, 2=Major 2nd, 3=minor 3rd

4=Major 3rd, 5=Perfect 4th, 6=Tri-tone

for example: 2,5,3,1


This option determines the order of ascending and descending applied to

the sequence of intervals entered.


-Ordered systematically applies every permutation starting with all ascending intervals,

and then proceeds through the permuations sequentially until ending with all descending.

-Shuffled uses every permutation available but in random order.

-Random uses random permuations in random order with possible repetition.


Note: The provided permuations use the last pitch from the previous line to start the next

permutation and because of this the entire list consists of only of the interval pattern.
